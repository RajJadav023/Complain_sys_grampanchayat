# 🏛️ Digital Complaint Management System for Grampanchayat

A full-stack web application designed to digitize the complaint management process in Grampanchayats. 
Citizens can register complaints, track their status, and access important government information,
while administrators can efficiently manage complaints through a dedicated dashboard.

---

## ❓ Problem

Traditional grievance redressal systems in Grampanchayats rely on manual, paper-based processes, which often lead to:

- ⏳ Delays in complaint resolution
- 🔍 Lack of transparency and complaint tracking
- 🧾 Increased administrative workload
- 📚 Difficulty in maintaining records

---

## 💡 Solution

This application provides:

- 🌐 Online complaint registration and tracking
- 🧑‍💼 Admin dashboard for complaint management
- 📲 Real-time complaint status updates
- 📰 Community news & event announcements
- 📄 Government document guidance
- 🌍 Multilingual support
- 🛡️ Improved transparency and faster grievance resolution

---

## 🚀 Features

### 👤 User
- User Registration & Login
- Automatic Village Detection using PIN Code API
- Submit & Track Complaints
- Upload Complaint Images/Documents
- Community News & Events
- Government Document Guidance
- Multilingual Support

### 🧑‍💼 Admin
- Manage Complaints
- Update Complaint Status
- Respond to Users
- Publish Community News

---

## 📝 Complaint Categories

## 🛠️ Tech Stack

- **Frontend:** React.js, HTML5, CSS3, JavaScript
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **API:** PIN Code API
- **Tools:** Git, GitHub

---

## ▶️ Getting Started

1. Clone the repository

```bash
git clone https://github.com/RajJadav023/Complaint_sys_grampanchayat.git
```

2. Install dependencies

```bash
npm install
```

3. Start the application

```bash
npm start
```
